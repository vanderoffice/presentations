The Solution: Technical Executive Assignment

Create a new executive classification requiring verified technical competence for IT leadership positions.

Covered Positions

  • State & Agency CIOs
  • State & Agency CISOs
  • State & Agency CDOs
  • State & Agency CTOs
  • IT Director-level positions

What Changes

  • Technical competence verification required
  • Appointing authorities must document assessment
  • Independent review for positions >$200K
  • Public transparency dashboard
Legal Precedent: Government Code ¬ß19889 already creates technical exceptions for physicians, attorneys, engineers, and scientists. This proposal extends that logic to IT roles.

2

10+ years technical leadership
Documented outcomes (projects, teams, budgets)
References from technical peers

3

Independent Panel Review

Three-person panel (CalHR, CDT, external)
Structured technical interview
Written recommendation (public record)

Panel review accommodates non-traditional paths: military cyber operators, startup CTOs, career changers.
